Module: Thumbs

The Thumbs module is responsible for rendering showing and hiding vtt pictures based on mousing mouse on progress bar.
Parameters:
Name Type Description
vidElement the this.video element
vttThumbsTrackIndex the index of Vtt thumbs track in textTracks list of this.video
thumbImgElement the img element used for thumbs
thumbImgElement the div element used for thumbs
progressBarElement the progress element used for thumbs
